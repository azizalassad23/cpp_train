1. PRINT MENGGUNAKAN std::cout

Dalam C++, kita membutuhkan pustaka <iostream> untuk mencetak sesuatu ke layar. Perintah dasarnya adalah std::cout diikuti oleh tanda <<.

Syarat utama: Teks (string) wajib diapit tanda kutip ganda " " dan setiap baris perintah diakhiri dengan titik koma ;.

> LATIHAN 1

Ubah kode di editor sebelah kanan agar mencetak teks: Sistem Aktif!

2. PRINT BILANGAN & BARIS BARU

Untuk mencetak angka (integer), kalian tidak perlu menggunakan tanda kutip. Komputer membaca angka murni berbeda dengan teks.

Selain itu, untuk memindahkan teks ke baris bawah (Enter), gunakan \n di dalam tanda kutip teks kalian.

> LATIHAN 2

Di bawah baris kode Latihan 1, tambahkan perintah baru untuk mencetak angka 2025.

Pastikan Latihan 1 diakhiri dengan \n agar angka 2025 berada di baris baru.

3. OPERASI BILANGAN (EKSPRESI)

C++ bisa langsung menghitung matematika di dalam std::cout. Gunakan simbol + (tambah), - (kurang), * (kali), dan / (bagi).

Contoh: std::cout << 10 * 5; akan langsung mencetak angka 50 di layar terminal.

> LATIHAN 3

Cetak hasil dari operasi matematika: 100 + 50 * 2.

4. STUDI KASUS: MERETAS SERVER GRID

Kalian menemukan sebuah brankas data rahasia. Untuk membukanya, sistem meminta dua baris kode sandi berupa luas dan keliling dari ruangan Server Grid Sektor 7 yang memiliki panjang 364 meter dan lebar 79 meter.

Bantulah meretas sistem ini dengan membuat program C++ yang menghitung:

  • Baris pertama: Luas Server Grid (hasil perkalian panjang dan lebar).
  • Baris kedua: Keliling Server Grid (dua kali dari hasil penjumlahan panjang dan lebar).
