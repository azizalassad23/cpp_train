STATUS: STANDBY

1. PENGGUNAAN namespace std

Mengetik std:: di setiap baris perintah tentu sangat merepotkan. Untuk menyederhanakannya, kita bisa mendeklarasikan using namespace std; tepat di bawah #include <iostream>.

Dengan perintah tersebut, komputer sudah tahu bahwa kita menggunakan pustaka standar. Kalian cukup mengetik cout << "Teks"; tanpa embel-embel std:: lagi!

> LATIHAN 1

Perhatikan editor di kanan, using namespace std; sudah disiapkan. Cetaklah teks berikut menggunakan cout yang lebih singkat: Namespace Aktif!

2. PERKENALAN VARIABEL (INT)

Variabel adalah "kotak penyimpanan" di dalam memori. Sebelum menggunakannya, kita harus memesan tipe kotaknya.

Untuk menyimpan angka bulat, gunakan tipe int (Integer). Cara membuatnya: int nama_kotak = isinya;

> LATIHAN 2

Hapus kode Latihan 1. Buatlah sebuah variabel int bernama tahun dan beri nilai 2077. Lalu cetak variabel tersebut (ingat, mencetak variabel tidak memakai tanda kutip).

Output yang diharapkan: 2077

3. MEMPERBARUI NILAI VARIABEL

Isi dari variabel bisa diubah (di-update) kapan saja dengan operasi matematika. Saat mengubah nilainya, kalian tidak perlu menulis int lagi.

Contoh update: tahun = tahun + 3; (Nilai tahun yang baru adalah nilai tahun yang lama ditambah 3).

> LATIHAN 3

Lanjutkan kodemu dari Latihan 2. Tambahkan \n saat mencetak 2077. Kemudian di baris bawahnya, perbarui variabel tahun menjadi 2080, lalu cetak lagi.

Output yang diharapkan:
2077
2080

4. TIPE DATA STRING

Jika int untuk angka, maka tipe data string digunakan untuk menyimpan teks (huruf/kalimat).

Syaratnya, kalian wajib menambahkan #include <string> di baris paling atas agar mesin mengenali kotak teks ini.

> LATIHAN 4

Hapus kode di dalam main(). Buatlah variabel string bernama senjata dengan isi "Katana" (gunakan tanda kutip). Lalu cetak variabel tersebut.

Output yang diharapkan: Katana

5. STUDI KASUS: LOGISTIK PASUKAN

Markas perlawanan saat ini memiliki 63 Drone dan 192 Android. Panglima akan melakukan rotasi logistik selama dua fase operasi:

Fase 1:
- Markas memproduksi Android baru sebanyak total Drone yang ada, lalu...
- Sepertiga (1/3) dari total Drone dihancurkan.

Fase 2:
- Markas memproduksi Drone baru sebanyak total Android yang ada, lalu...
- 10 Android ditarik dari peredaran (dikurangi).

> UJIAN AKHIR BAB 2

Hapus semua kode latihan sebelumnya. Buatlah variabel int drone = 63; dan int android = 192;. Lakukan pembaruan nilai variabel (update) selangkah demi selangkah sesuai cerita operasi di atas dengan operator matematika!

Hint Fase 1: android = android + drone; lalu di bawahnya drone = drone - (drone / 3);

Berapakah total gabungan Drone dan Android saat ini? Cetak hasil penjumlahan akhir mereka!
