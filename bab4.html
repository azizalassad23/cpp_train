BAB 4: PERCABANGAN (IF-ELSE)

STATUS: STANDBY

1. PENGENALAN IF

Dalam C++, if (jika) digunakan untuk membuat keputusan. Kode di dalam tanda kurung kurawal { } hanya akan dijalankan JIKA kondisinya bernilai benar (true).

> LATIHAN 1

Buat variabel int level = 10;.
Buatlah struktur: if (level > 5) { ... }.
Di dalam tanda kurung kurawal tersebut, cetaklah teks: Akses Diberikan

2. OPERASI MODULO (%)

Modulo (disimbolkan dengan %) adalah operasi untuk mencari sisa hasil bagi. Ini sangat sering dipakai untuk mengecek apakah suatu angka genap atau ganjil.

Contoh: 10 % 3 hasilnya adalah 1 (karena 10 dibagi 3 adalah 9, sisanya 1). Sedangkan 10 % 2 hasilnya 0 (karena habis dibagi).

> LATIHAN 2

Hapus kode Latihan 1. Buat variabel int kode = 7;. Cetak hasil dari kode % 2 menggunakan cout.

Output yang diharapkan: 1

3. PERKENALAN ELSE

Bagaimana jika kondisi if bernilai salah (false)? Di sinilah kita menggunakan else (jika tidak). else adalah rencana cadangan jika if gagal memenuhi syarat.

> LATIHAN 3

Hapus kodemu. Buat int level = 3;.
Gunakan if (level > 5) untuk mencetak "Akses Diberikan".
Lalu tambahkan blok else { ... } di bawahnya untuk mencetak Akses Ditolak.

Output yang diharapkan: Akses Ditolak (karena 3 tidak lebih besar dari 5).

4. PERKENALAN ELSE IF

Jika kita memiliki banyak kondisi bertingkat, gunakan else if. Program akan mengecek dari atas ke bawah, dan berhenti begitu menemukan satu kondisi yang benar.

> LATIHAN 4

Buat int poin = 75;. Buat percabangan:
- Jika poin >= 90 cetak "Gold"
- Else if poin >= 70 cetak Silver
- Else cetak "Bronze"

Output yang diharapkan: Silver

5. PENDALAMAN EKSPRESI BOOLEAN

Untuk mengecek kondisi ganda, kita menggunakan operator logika:
- && (AND): Keduanya harus benar.
- || (OR): Salah satu saja benar, maka diterima.
- == : Apakah SAMA DENGAN?
- != : Apakah TIDAK SAMA DENGAN?

> LATIHAN 5

Buat int pin = 1234; dan int rfid = 99;.
Buat if dengan kondisi: (pin == 1234 && rfid == 100) untuk mencetak "Masuk". Tambahkan else untuk mencetak Blokir.

Output yang diharapkan: Blokir (karena RFID salah).
