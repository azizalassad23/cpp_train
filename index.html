[01] ARSIP_PENGETAHUAN: TENTANG C++

> SEJARAH_SISTEM

C++ dikembangkan pertama kali oleh Bjarne Stroustrup pada tahun 1979 di Bell Labs sebagai ekstensi dari bahasa C (awalnya disebut "C with Classes"). Bahasa ini dirancang untuk menggabungkan performa tingkat rendah dari bahasa C dengan paradigma pemrograman tingkat tinggi. C++ distandarisasi secara global pada tahun 1998 dan terus menjadi tulang punggung teknologi modern.

> PENGGUNAAN_UTAMA

  • Embedded Systems & IoT: Sangat efisien untuk memprogram mikrokontroler (seperti NodeMCU ESP8266 / ESP32) yang memiliki sumber daya memori terbatas untuk proyek robotika dan sensor.
  • Pengembangan Game: Sebagian besar game engine kelas AAA (seperti Unreal Engine) dibangun menggunakan C++ untuk render grafik berkecepatan tinggi.
  • Sistem Operasi & Browser: Bagian inti dari Windows, MacOS, serta mesin di balik Google Chrome (V8 Engine) ditulis dalam C++.

> SILABUS_DASAR (BASIC)

Fase awal di C++ Academy akan melatih algoritma fundamental Anda, meliputi: Sintaks Output, Deklarasi Variabel & Tipe Data, Operasi Matematika, Percabangan (If-Else), dan Konstruksi Perulangan (While & For) untuk memecahkan masalah logika dasar.

> SILABUS_LANJUTAN (ADVANCED)

Setelah lulus fase dasar, agen akan menghadapi tantangan arsitektur tingkat tinggi seperti: Fungsi (Functions), Array & Matriks, Pointer & Manajemen Memori, serta Pemrograman Berorientasi Objek (OOP).
