BAB 6: ARRAY (PENYIMPANAN MATRIKS)

STATUS: STANDBY

1. PENGENALAN ARRAY

Array adalah sebuah loker penyimpanan panjang yang bisa menyimpan banyak data dengan tipe yang sama dalam satu nama variabel. Ingat: Indeks array di C++ selalu dimulai dari 0, bukan 1.

> LATIHAN 1

Buat array: int node[3] = {10, 50, 99};
Cetak isi loker kedua (angka 50). Hint: Gunakan indeks [1].

Output: 50

2. PASAR GELAP NODE (Toko Kandang I)

Daripada mencetak array satu per satu (node[0], node[1]), kita bisa mengkombinasikan Array dengan for loop untuk memindai seluruh data secara instan!

> LATIHAN 2

Buat array: int kapasitas[5] = {10, 20, 30, 40, 50};. Gunakan for loop (dari i=0 sampai 4) untuk mencetak semuanya. Beri spasi antar angka.

Output: 10 20 30 40 50

3. KAPASITAS TERBESAR (Kandang Terbesar)

Untuk mencari data terbesar di dalam matriks, kita butuh variabel sementara (misal: max). Bandingkan isi array satu per satu menggunakan if di dalam loop.

> LATIHAN 3

Diberikan array: int daya[5] = {15, 42, 7, 99, 23};.
Gunakan loop untuk mencari dan mencetak angka yang paling besar.

Output: 99

4. LALU LINTAS TRANSAKSI (Jual Beli Bebek)

Seringkali kita harus mengakumulasikan seluruh data dalam array. Siapkan variabel penyimpan, misalnya int total = 0; lalu tambahkan isi array ke dalamnya satu per satu.

> LATIHAN 4

Diberikan array: int transfer[3] = {100, 200, 300};.
Hitung jumlah total dari ketiga transaksi tersebut dengan loop, lalu cetak hasilnya.

Output: 600

5. TOGGLE SAKLAR MATRIKS (Bermain Lampu)

Sebuah barisan node keamanan bisa disimbolkan dengan 1 (Aktif) dan 0 (Mati). Terkadang agen harus meretasnya dengan membalik kondisinya (Toggle).

> LATIHAN 5

Diberikan array saklar: int saklar[5] = {1, 0, 1, 1, 0};.
Ubah yang 1 menjadi 0, dan yang 0 menjadi 1 menggunakan if-else di dalam loop. Cetak hasil array yang baru dengan spasi.

Output: 0 1 0 0 1

6. MELACAK NODE (Toko Kandang II)

Ini disebut Linear Search (Pencarian Linear). Kalian harus menyisir array dari ujung ke ujung untuk menghitung seberapa sering sebuah nilai anomali muncul.

> LATIHAN 6

Diberikan array log: int log_error[5] = {12, 404, 89, 404, 77};.
Hitunglah ada berapa banyak angka 404 di dalam log tersebut. Cetak jumlahnya.

Output: 2

7. MENGHAPUS JEJAK (Menutup Toko)

Agen yang handal harus bisa memutarbalikkan jalur data untuk menutupi jejak pelarian. Caranya? Jalankan for loop secara mundur (dari indeks terbesar ke terkecil).

> LATIHAN 7

Diberikan jejak: int jejak[5] = {1, 2, 3, 4, 5};.
Cetak array tersebut secara terbalik. Hint: Mulai loop dari i = 4, turun ke i >= 0, gunakan i--.

Output: 5 4 3 2 1
